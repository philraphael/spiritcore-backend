SpiritVerse Premium Game Asset Pack

What this pack is:
- A Codex-ready structured crop pack derived from the premium concept sheet.
- Best used for fast implementation, theming, prototypes, splash/backdrop use, UI frames, and premium placeholder assets.

What this pack is not yet:
- Not true transparent sprite sheets for every individual piece.
- Not frame-by-frame animation atlases.
- Not final production-cut isolated 3D assets for all games.

Recommended Codex usage:
1. Use boards/* as scene or board bases.
2. Use ui/* for banners, frame concepts, and state overlays.
3. Use tokens/* and pieces/* as temporary premium token sheets to slice or replace later.
4. Use fx/* for glow/portal/highlight overlays.
5. Use ships/* as a premium battleship visual reference set.
