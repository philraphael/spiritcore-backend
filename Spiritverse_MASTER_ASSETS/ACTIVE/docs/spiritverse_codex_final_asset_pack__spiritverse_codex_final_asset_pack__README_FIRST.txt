SpiritVerse Codex Final Asset Pack

This pack is structured for immediate Codex implementation.

ROOT PURPOSE
- Upgrade current game visuals with premium themed boards, pieces, tokens, ships, UI banners, and FX.
- Use boards as active surfaces or framed backplates.
- Use lineup/set assets when fully isolated sprites are not yet available.

FOLDERS
- boards/: board bases and alternates
- pieces/: chess/checkers premium piece lineups
- tokens/: connect four, go, tic tac toe token sets
- ships/: battleship fleets
- ui/: banners, modals, turn and outcome frames
- fx/: highlight rings, portal/beam/crosshair effects
- rooms/: room/backdrop art per game theme
- concepts/: reference scenes and full sheets

IMPLEMENTATION PRIORITY
1. Chess
2. Checkers
3. Connect Four
4. Battleship
5. Tic Tac Toe
6. Go (preview or full later)

HONEST LIMITATION
These assets are implementation-ready premium crops and references, but not yet a perfect final production sprite library. For final perfection, a future pass should add transparent isolated per-piece/per-ship/per-token exports and animation-ready layers.
